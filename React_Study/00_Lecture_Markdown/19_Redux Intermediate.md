# 19_Redux Intermediate

### Redux로 비동기 호출 다루기

리듀서 함수는 순수함수이며, 사이드 이펙트 없이 동기적으로 작동되는 함수여야함

Input(Old State + Action) => OutPut(New State)

★☆ 절대 리듀서 함수 내부에서 부수효과를 내는 행동을 하지 말 것. 그것이 동기식이든, 비동기식이든 절대 내부에서 수행하지 말 것!!☆★
