# 00_Getting_started

## 1. React 시작하기

### 1) Node.js 설치

https://nodejs.org/ko

<br>

### 2) React-App 설치

#### (1) React-App 설치

```bash
npm install -g create-react-app
```

#### (2) React-App 제거

```bash
npm uninstall -g create-react-app
```

<br>

### 3) React-App 생성 및 실행

```bash
npx create-react-app [앱 이름]
cd [앱 이름]
npm start
```
